## 1. 자유게시판 목록 화면

### 1.1 게시판 기본 정보 조회

**요청 데이터**

- 게시판 ID

**응답 데이터**

- 게시판 제목
- 학교 이름
- 고정/추천 게시글 (리스트)
    - 게시글 ID
    - 고정/추천 게시글 유형 (질문글, 기숙사 등)
    - 게시글 제목
    - 게시글 내용
- HOT 인기 게시글 제목

**검증 조건**

- 유효한 게시판 ID가 아닌 경우 에러

### 1.2 게시글 피드 조회

**요청 데이터**

- 게시판 ID
- 페이지네이션 정보

**응답 데이터**

- 게시글 피드 (리스트)
    - 게시글 ID
    - 게시글 제목
    - 게시글 내용
    - 댓글 수
    - 공감 수
    - 작성 시각
    - 글쓴이
    - 썸네일 사진
- 페이지네이션 정보

**검증 조건**

- 유효한 게시판 ID가 아닌 경우 에러

## 2. 게시글 상세 화면

### 2.1 게시글 상세 정보 조회

**요청 데이터**

- 게시판 ID, 게시글 ID

**응답 데이터**

- 게시글 제목
- 게시글 내용
- 글쓴이
- 게시글 사진 리스트
- 댓글 수
- 공감 수
- 스크랩 수
- 작성 시각
- 로그인한 유저가 해당 게시글에 공감을 눌렀는지 여부
- 로그인한 유저가 해당 게시글을 스크랩했는지 여부

**검증 조건**

- 유효한 게시판 ID, 게시글 ID가 아닌 경우 에러

### 2.2 댓글 목록 조회

**요청 데이터**

- 게시판 ID, 게시글 ID
- 페이지네이션 정보

**응답 데이터**

- 댓글 목록 (리스트)
    - 댓글 ID
    - 글쓴이
    - 댓글 내용
    - 작성 시각
    - 댓글 공감 수
    - 대댓글 (리스트)
        - 댓글 ID
        - 글쓴이
        - 댓글 내용
        - 작성 시각
        - 댓글 공감 수
- 페이지네이션 정보

**검증 조건**

- 유효한 게시판 ID, 게시글 ID가 아닌 경우 에러

### 2.3 댓글 등록

**요청 데이터**

- 작성한 유저 정보
- 댓글을 다는 게시판 ID, 게시글 ID, 부모 댓글 ID (대댓글인 경우)
- 댓글 내용
- 익명 여부

**응답 데이터**

- 댓글 ID
- 글쓴이
- 댓글 내용
- 작성 시각
- 익명 여부

**검증 조건**

- 유효한 게시판 ID, 게시글 ID, 부모 댓글 ID가 아닌 경우 에러
- 댓글 내용이 비어있으면 에러
- 인증된 유저가 아닌 경우 에러

### 2.4 게시글 삭제

**요청 데이터**

- 게시판 ID, 게시글 ID

**응답 데이터**

- 삭제된 게시글 ID

**검증 조건**

- 유효한 게시판 ID, 게시글 ID가 아닌 경우 에러
- 작성자와 삭제를 요청한 유저가 일치하지 않는 경우 에러

### 2.5 게시글 좋아요 / 스크랩

**요청 데이터**

- 게시판 ID, 게시글 ID

**응답 데이터**

- 현재 공감 상태 / 스크랩 상태
- 스크랩 수 / 공감 수

**검증 조건**

- 유효한 게시판 ID, 게시글 ID가 아닌 경우 에러
- 인증된 유저가 아닌 경우 에러

## 3. 글 쓰기 화면

### 3.1 게시글 등록

**요청 데이터**

- 작성한 유저 정보
- 작성하는 게시판 ID
- 제목
- 본문
- 익명 여부
- 질문 여부
- 게시글 사진 리스트

**응답 데이터**

- 게시글 ID
- 제목
- 본문
- 글쓴이
- 질문 여부
- 게시글 사진 리스트
- 작성 시각

**검증 조건**

- 유효한 게시판 ID가 아닌 경우 에러
- 제목 필드가 비어있거나 50자 초과 시 에러
- 본문이 비어있으면 에러
- 인증된 유저가 아닌 경우 에러

### 3.2 게시글 수정

**요청 데이터**

- 작성한 유저 정보
- 게시판 ID, 게시글 ID
- 제목
- 본문
- 익명 여부
- 질문 여부
- 게시글 사진 리스트

**응답 데이터**

- 게시글 ID
- 제목
- 본문
- 글쓴이
- 질문 여부
- 게시글 사진 리스트
- 수정 시각

**검증 조건**

- 유효한 게시판 ID, 게시글 ID가 아닌 경우 에러
- 작성자와 수정을 요청한 유저가 일치하지 않는 경우 에러
- 제목 필드가 비어있거나 50자 초과 시 에러
- 본문이 비어있으면 에러
